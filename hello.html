Developed a Web Application as an extension for the Southern CT state university banner website.

SCSU Student Waiting list option Web App.

As a group from my software development group we created a web application which extends the website banner registration page which implements a waiting list option for students looking to get into a class that has already been filled up to its' maximum capacity. The three main funtions are as follows:
First, it is designed to help students meet their requirements for graduation. Our software is designed to allow seniority importance by implementing a waitlist prioritized towards Seniors. Our software will also implement a notification system, such that if a required class has an opening the student will be immediately notified of the opening. We also notify students in case the class cancels.
Second, it allows faculty to see what majors are showing interest in classes they will teach so as to help prepare for the class.
Third, our software keeps track of how much interest a particular class is receiving so an administrator could determine if another section of the class needs to be opened.
In working with this project I had to think about all case scenarios for both students and faculty so that there will be no room for errors. As a team we were productive using the Agile methodology therefore we discussed functions, ideas and roles before even thinking about the user interface or how we would implement the features.
The making of this project involved the following technologies:
Python, JavaScript, HTML, CSS, MYSQL, Google Developmemt Console(GDP), Flask, Jinja, linux command prompt, git, GitHub, Slack and Trello in an Agile environment.
